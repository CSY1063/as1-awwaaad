Introduction

In this report, I will discuss the process of designing a portfolio with only HTML and CSS.
I will reflect on the challenges faced, the skills acquired and the lessons learned throughout the design process

Design Process

I did not have any issues with the deisgn process as I am a UI Designer. The design process of creating my portfolio website involved several key stages:

  • Planning: I started by defining the purpose of the portfolio website and setting specific goals. I outlined the sections I wanted to include, such as Home, Contact, Projects and so on.
  • Research and Inspiration: I conducted research on existing portfolio websites to gather inspiration for design layouts, color schemes and content organization. This helped me generate ideas for my own website.
  • Wireframing and Prototyping: Before diving into coding, I created wireframes and prototypes to visualize the layout and structure of each webpage. This step helped me plan the placement of elements and the overall user flow.
  • HTML Markup: With the wireframes as a guide, I began coding the HTML structure of the website. I focused on creating semantic markup and organizing content into meaningful sections and divs.

    • Skills Acquired

    Despite the challenges, the project helped me develop several valuable skills:

  • HTML and CSS Proficiency: Through hands-on practice, I improved my skills in HTML markup and CSS styling, gaining a better understanding of how to structure web pages and apply styles effectively. I now know how to use different HTML tags and styling methods.
  • Responsive Design Techniques: I learned various techniques for creating responsive layouts, including media queries to ensure the website was accessible and functional across different devices. Although I still need to practice how to make my desgigns responsive on different resolutions.
  • Problem-Solving Skills: The project involved solving complex problems and overcoming technical challenges, which improved my problem-solving skills and critical thinking abilities.
  • Attention to Detail: Paying attention to small details such as spacing, alignment and consistency was important for creating a polished and professional-looking website.

    • Lessons

    Doing this assignment opened my mind to alot of things about web designing as a novice and taught me several valuable lessons:

  • Continuous Learning: Web design and development are constantly evolving fields, requiring continuous learning and adaptation to stay updated with the latest trends and technologies.
  • Simplicity is Key: Keeping the design simple and focused is essential for creating an effective portfolio website. Avoiding clutter and unnecessary elements helps in conveying key information more effectively.
  • Feedback is Valuable: Seeking feedback from peers, mentors and users is crucial for improving the design and usability of the website. Constructive feedback helps in identifying areas for improvement and refining the design further.
  • Persistence Pays Off: Overcoming challenges and learning from mistakes are essential parts of the design process. Persistence and dedication are key to achieving success in web design and development.

    • Conclusion

    Designing a portfolio website as a student was a challenging yet rewarding experience. It allowed me to apply and enhance my HTML and CSS skills while learning valuable design principles and techniques. Moving forward, I will continue to refine and update my portfolio website, incorporating new skills and experiences gained along the way. Overall, this project has been instrumental in my growth as a web designer and developer.
